# 📈 Regressão Linear – Preço vs Área de Casas

Este projeto implementa uma regressão linear simples do zero utilizando Python e plota os resultados com Matplotlib, incluindo estatísticas e interação com os pontos do gráfico.

## 🎯 Objetivo

Analisar a relação entre:

* Área das casas (m²)
* Preço das casas

E ajustar uma reta de regressão linear:

y = ax + b

## 📊 Funcionalidades

* Cálculo manual da regressão linear
* Plot dos dados e reta de regressão
* Box com estatísticas: Média, Variância, Desvio padrão
* Tooltip interativo ao clicar nos pontos

## 🛠️ Tecnologias

* Python
* NumPy
* Matplotlib

## ▶️ Como executar

```bash
python regressao_linear.py
```

ou 

```bash
make run
```

## 🧮 Fórmulas utilizadas

Coeficiente angular:

```
a = Σ((x - x̄)(y - ȳ)) / Σ((x - x̄)²)
```

Coeficiente linear:

```
b = ȳ - a x̄
```
