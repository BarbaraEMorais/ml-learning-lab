# Machine Learning Study Lab

## 🇧🇷 Português

Repositório dedicado ao estudo e implementação de modelos de Machine Learning com foco no entendimento matemático e na compreensão detalhada das etapas do processo.  
O objetivo é explorar os algoritmos em baixo nível, implementando-os do zero sempre que possível, para aprofundar conceitos como regressão, otimização, métricas e análise estatística.

Este repositório tem caráter educacional e experimental, servindo como laboratório para:
- Estudo de fundamentos de Machine Learning
- Implementação manual de algoritmos
- Visualização de dados e resultados
- Análise estatística e matemática
- Experimentação com diferentes abordagens

## 🇺🇸 English

Repository dedicated to studying and implementing Machine Learning models with a strong focus on mathematical understanding and low-level comprehension of each step in the process.  
The goal is to explore algorithms from scratch whenever possible, deepening knowledge of regression, optimization, metrics, and statistical analysis.

This repository is intended as an educational and experimental lab for:
- Studying Machine Learning fundamentals
- Manual implementation of algorithms
- Data visualization and analysis
- Statistical and mathematical exploration
- Experimenting with different approaches
